## Installation

You can install the package via composer:

```bash
composer require seebaermichi/unit-conversions
```

## Usage

``` php
$skeleton = new Seebaermichi\UnitConversions();
echo $skeleton->echoPhrase('Hello, Seebaermichi!');
```
